# v0.1.0 - In-memory queue via REPL

Release date: 2026-06-12

## Summary

This release introduces the first usable version of the message broker: a command-line REPL that stores messages in an in-memory queue.

## Added

- REPL command parsing
- In-memory queue using `VecDeque`
- `enqueue <message>` command
- `dequeue` command
- `list` command
- `exit` command

## Notes

This version does not include persistence, acknowledgements, retries, networking, or worker processing yet. and the payload for now is just a string
